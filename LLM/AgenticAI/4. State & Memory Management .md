## State & Memory Management (in Front Office Context)
Implement robust systems for tracking agent progress, learning from interactions, and maintaining context across workflows.

Front-office context means **real stakes** — trading desks, client advisory, sales, deal execution. Memory isn't just convenience, it's correctness.

---

### The 4 Memory Layers

```
┌─────────────────────────────────────────────────────┐
│                    AGENT BRAIN                      │
│                                                     │
│  ┌─────────────┐  ┌─────────────┐  ┌────────────┐  │
│  │  In-Context │  │  Episodic   │  │  Semantic  │  │
│  │   Memory    │  │   Memory    │  │   Memory   │  │
│  │ (right now) │  │  (history)  │  │  (facts)   │  │
│  └─────────────┘  └─────────────┘  └────────────┘  │
│                    ┌─────────────┐                  │
│                    │ Procedural  │                  │
│                    │  Memory     │                  │
│                    │ (how-to's)  │                  │
│                    └─────────────┘                  │
└─────────────────────────────────────────────────────┘
```

| Layer | What it stores | Lifespan | Storage |
|---|---|---|---|
| **In-Context** | Current task, recent messages | Single session | LLM context window |
| **Episodic** | Past interactions, decisions made | Long-term | Vector DB |
| **Semantic** | Facts about clients, positions, products | Long-term | Vector DB / SQL |
| **Procedural** | Successful workflows, tool-use patterns | Long-term | Prompt templates / fine-tune |

### Full Implementation

```python
from langgraph.graph import StateGraph, END
from langchain_openai import ChatOpenAI, OpenAIEmbeddings
from langchain_community.vectorstores import Chroma
from langchain_core.documents import Document
from typing import TypedDict, List, Optional, Annotated
from datetime import datetime
import json
import operator

llm = ChatOpenAI(model="gpt-4o")
embeddings = OpenAIEmbeddings()

# ─────────────────────────────────────────────
# 1. MEMORY STORES
# ─────────────────────────────────────────────

# Episodic: past interactions
episodic_store = Chroma(
    collection_name="episodic_memory",
    embedding_function=embeddings
)

# Semantic: client/market facts
semantic_store = Chroma(
    collection_name="semantic_memory",
    embedding_function=embeddings
)

class MemoryManager:
    """Unified interface for all memory operations."""

    # --- EPISODIC ---
    def save_episode(self, agent_id: str, task: str, outcome: str, metadata: dict):
        doc = Document(
            page_content=f"Task: {task}\nOutcome: {outcome}",
            metadata={
                "agent_id": agent_id,
                "timestamp": datetime.utcnow().isoformat(),
                "success": metadata.get("success", False),
                **metadata
            }
        )
        episodic_store.add_documents([doc])

    def recall_episodes(self, query: str, agent_id: str, k: int = 3) -> List[str]:
        results = episodic_store.similarity_search(
            query,
            k=k,
            filter={"agent_id": agent_id}
        )
        return [r.page_content for r in results]

    # --- SEMANTIC ---
    def save_fact(self, entity: str, fact: str, category: str):
        doc = Document(
            page_content=fact,
            metadata={
                "entity": entity,
                "category": category,   # "client" | "market" | "product"
                "updated_at": datetime.utcnow().isoformat()
            }
        )
        semantic_store.add_documents([doc])

    def recall_facts(self, query: str, category: str = None, k: int = 5) -> List[str]:
        filter_dict = {"category": category} if category else {}
        results = semantic_store.similarity_search(query, k=k, filter=filter_dict)
        return [r.page_content for r in results]

    # --- PROCEDURAL ---
    def get_procedure(self, task_type: str) -> str:
        procedures = {
            "client_inquiry": """
                1. Retrieve client profile and recent interactions
                2. Check current portfolio positions
                3. Pull relevant market data
                4. Generate personalized response
                5. Log interaction to episodic memory
            """,
            "trade_execution": """
                1. Validate client mandate and risk limits
                2. Check market conditions
                3. Run pre-trade compliance check
                4. Execute or escalate
                5. Confirm and log outcome
            """,
            "risk_alert": """
                1. Retrieve full position context
                2. Assess breach severity
                3. Identify remediation options
                4. Escalate if threshold exceeded
                5. Document decision trail
            """
        }
        return procedures.get(task_type, "No procedure found.")

memory = MemoryManager()

# ─────────────────────────────────────────────
# 2. AGENT STATE
# ─────────────────────────────────────────────
class FrontOfficeState(TypedDict):
    # Task
    task_id: str
    task_type: str                          # "client_inquiry" | "trade_execution" | "risk_alert"
    raw_input: str

    # Memory layers loaded into context
    episodic_context: List[str]             # relevant past episodes
    semantic_context: List[str]             # relevant facts
    procedural_context: str                 # workflow steps to follow

    # Working state
    client_id: Optional[str]
    current_analysis: str
    decision: str
    confidence: float                       # 0.0 - 1.0
    requires_escalation: bool

    # Audit trail — append-only
    audit_log: Annotated[List[str], operator.add]

    # Output
    response: str
    status: str                             # "complete" | "escalated" | "failed"

# ─────────────────────────────────────────────
# 3. NODES
# ─────────────────────────────────────────────

def load_memory(state: FrontOfficeState) -> FrontOfficeState:
    """Load all relevant memory layers before processing."""

    query = state["raw_input"]
    agent_id = state["task_type"]

    # Pull from all memory layers
    episodes = memory.recall_episodes(query, agent_id=agent_id, k=3)
    facts = memory.recall_facts(query, k=5)
    procedure = memory.get_procedure(state["task_type"])

    log_entry = f"[{datetime.utcnow().isoformat()}] Memory loaded: {len(episodes)} episodes, {len(facts)} facts"

    return {
        "episodic_context": episodes,
        "semantic_context": facts,
        "procedural_context": procedure,
        "audit_log": [log_entry]
    }

def analyze(state: FrontOfficeState) -> FrontOfficeState:
    """Core reasoning node — uses all memory context."""

    prompt = f"""
    You are a front-office AI agent. Use all context below to analyze the request.

    TASK: {state['raw_input']}
    TASK TYPE: {state['task_type']}

    PROCEDURE TO FOLLOW:
    {state['procedural_context']}

    RELEVANT PAST EPISODES:
    {chr(10).join(state['episodic_context']) or 'None'}

    KNOWN FACTS:
    {chr(10).join(state['semantic_context']) or 'None'}

    Respond in JSON:
    {{
        "analysis": "your reasoning",
        "decision": "recommended action",
        "confidence": 0.0-1.0,
        "requires_escalation": true/false,
        "escalation_reason": "if applicable"
    }}
    """

    result = llm.invoke(prompt)

    try:
        parsed = json.loads(result.content)
    except:
        parsed = {
            "analysis": result.content,
            "decision": "manual review required",
            "confidence": 0.0,
            "requires_escalation": True,
            "escalation_reason": "Failed to parse response"
        }

    log_entry = f"[{datetime.utcnow().isoformat()}] Analysis complete. Confidence: {parsed['confidence']}. Escalate: {parsed['requires_escalation']}"

    return {
        "current_analysis": parsed["analysis"],
        "decision": parsed["decision"],
        "confidence": parsed["confidence"],
        "requires_escalation": parsed["requires_escalation"],
        "audit_log": [log_entry]
    }

def generate_response(state: FrontOfficeState) -> FrontOfficeState:
    """Generate client-facing or internal response."""

    prompt = f"""
    Based on this analysis, generate a professional front-office response.

    Task: {state['raw_input']}
    Analysis: {state['current_analysis']}
    Decision: {state['decision']}

    Be concise, precise, and professional.
    """

    result = llm.invoke(prompt)
    log_entry = f"[{datetime.utcnow().isoformat()}] Response generated."

    return {
        "response": result.content,
        "status": "complete",
        "audit_log": [log_entry]
    }

def escalate(state: FrontOfficeState) -> FrontOfficeState:
    """Route to human — log full context for handoff."""

    handoff_package = {
        "task_id": state["task_id"],
        "task": state["raw_input"],
        "analysis": state["current_analysis"],
        "decision": state["decision"],
        "confidence": state["confidence"],
        "audit_trail": state["audit_log"]
    }

    log_entry = f"[{datetime.utcnow().isoformat()}] ESCALATED TO HUMAN. Package: {json.dumps(handoff_package)}"

    return {
        "response": f"Escalated to human reviewer. Task ID: {state['task_id']}",
        "status": "escalated",
        "audit_log": [log_entry]
    }

def save_memory(state: FrontOfficeState) -> FrontOfficeState:
    """Persist outcome back to episodic memory — agent learns from this run."""

    memory.save_episode(
        agent_id=state["task_type"],
        task=state["raw_input"],
        outcome=state["decision"],
        metadata={
            "success": state["status"] == "complete",
            "confidence": state["confidence"],
            "task_id": state["task_id"]
        }
    )

    log_entry = f"[{datetime.utcnow().isoformat()}] Episode saved to memory."

    return {"audit_log": [log_entry]}

# ─────────────────────────────────────────────
# 4. ROUTING
# ─────────────────────────────────────────────
def route_after_analysis(state: FrontOfficeState) -> str:
    """Escalate if low confidence or flagged."""
    if state["requires_escalation"] or state["confidence"] < 0.75:
        return "escalate"
    return "respond"

# ─────────────────────────────────────────────
# 5. BUILD GRAPH
# ─────────────────────────────────────────────
graph = StateGraph(FrontOfficeState)

graph.add_node("load_memory",       load_memory)
graph.add_node("analyze",           analyze)
graph.add_node("generate_response", generate_response)
graph.add_node("escalate",          escalate)
graph.add_node("save_memory",       save_memory)

graph.set_entry_point("load_memory")
graph.add_edge("load_memory", "analyze")

graph.add_conditional_edges(
    "analyze",
    route_after_analysis,
    {
        "respond":  "generate_response",
        "escalate": "escalate"
    }
)

graph.add_edge("generate_response", "save_memory")
graph.add_edge("escalate",          "save_memory")
graph.add_edge("save_memory",       END)

app = graph.compile()

# ─────────────────────────────────────────────
# 6. RUN
# ─────────────────────────────────────────────
result = app.invoke({
    "task_id": "TASK-001",
    "task_type": "client_inquiry",
    "raw_input": "Client Goldman account wants to increase EM equity exposure by 15%",
    "episodic_context": [],
    "semantic_context": [],
    "procedural_context": "",
    "client_id": "GS-001",
    "current_analysis": "",
    "decision": "",
    "confidence": 0.0,
    "requires_escalation": False,
    "audit_log": [],
    "response": "",
    "status": ""
})

print("Response:", result["response"])
print("Status:  ", result["status"])
print("\nAudit Trail:")
for entry in result["audit_log"]:
    print(" ", entry)
```

---

### Flow

```
START
  │
  ▼
load_memory          ← pulls episodic + semantic + procedural context
  │
  ▼
analyze              ← reasons with full memory context, scores confidence
  │
  ├── confidence >= 0.75 ──► generate_response ──► save_memory ──► END
  │
  └── confidence < 0.75  ──► escalate ──────────► save_memory ──► END
```

---

### Key Design Decisions for Front-Office

| Decision | Approach | Why |
|---|---|---|
| **Append-only audit log** | `Annotated[List, operator.add]` | Immutable decision trail for compliance |
| **Confidence threshold** | 0.75 escalation gate | Low-confidence = human review, non-negotiable |
| **Save memory after every run** | `save_memory` node always executes | Agent improves with every interaction |
| **Procedural memory** | Hardcoded workflows per task type | Enforces process compliance |
| **Structured JSON decisions** | Parsed from LLM output | Enables routing logic on confidence/escalation flags |

---

**One-liner summary:**
> "In front-office AI, memory isn't optional — every decision needs context from the past, facts about the present, and a traceable log for compliance. The agent must get smarter with every run, and never act autonomously when confidence is low."
